#Biodegradation Classification of Chemical Compounds 

This project applies machine learning to classify chemical compounds as ready-biodegradable or non-ready-biodegradable using the qsar-biodeg dataset.

## Methods
- K-Nearest Neighbors (KNN) 
- Support Vector Machine (SVM)
- Genetic Algorithm (GA) for feature selection
- Stratified 5-fold cross validation

## Dataset
- qsar-biodeg
- 1055 samples
- 41 molecular descriptors

## Files 
- biodegradation_classification.ipynb - baseline models
- classification_with_feature_selection.ipynb - feature selection and improved models
- genetic_algorithm_334.ipynb - genetic algorithm implementation
